# Cmark
Camera Log

- 为照片添加水印(拍摄信息)

- 目前支持 尼康，佳能, 索尼，松下

### Build

#### Build it in Windows
- Open folder in VS or Other IDE(must support CMake project)
- build it & run


#### Build it in Linux
- need `Qt5 , CMake , Clang Complier , ninja-build`(please install them)
- Run build.sh


### Run

#### 操作介绍

- 选择文件菜单中的打开选项
    - 选择文件夹打开，左侧视图将会预览当前文件夹下所有支持的图片文件
    - ![][previewFile]

- 点相应的文件 右侧窗口将会预览显示选择的文件
    - ![][nikon_secone]
